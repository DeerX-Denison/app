# Package Script

```
# Run android development
yarn android

# Run ios development
yarn ios

# Run web development
yarn web

# Run all
yarn start

# Run test once
yarn test

# Run test continually
yarn test:watch

# Lint code
yarn lint

# Format code
yarn format
```
